## API examples

!SLIDE code

### Access to API factory (Java)

Get the API factory: `GoodTool api = GoodTool.getAPI();`

!SLIDE code

### Usage of API factory (Java)

Set monitoring of only one CPU thread:
    GoodTool api = GoodTool.getAPI();
    api.setMonitoredThreads(1);
